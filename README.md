# 🧃 POKETA

**POKETA는 귀엽고 똑똑한 헬스 음료 브랜드**입니다.  
손바닥을 스캔하면 부족한 영양소를 분석해, **맞춤형 건강 음료**를 제공해요!  

---

## 📚 About Project

| 항목         | 설명                                                    |
|--------------|---------------------------------------------------------|
| 브랜드 이름  | **POKETA** (Pocket + Vita)                             |
| 컨셉         | 휴대하기 좋은 헬스 음료 브랜드                         |
| 주요 키워드  | 귀여움 · 맞춤형 · 건강 · 간편함 · 트렌디             |

이 웹사이트는 **POKETA의 브랜드 아이덴티티**를 시각적으로 표현하고,  
제품과 서비스를 소개하는 역할을 합니다.
